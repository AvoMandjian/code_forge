# References & Resources

## Official
*   **GitHub Repository**: [heckmon/code_forge](https://github.com/heckmon/code_forge) - Source code, issues, and discussions.
*   **Pub.dev Package**: [code_forge](https://pub.dev/packages/code_forge) - Installation instructions and versions.

## Dependencies
*   **re_highlight**: [pub.dev/packages/re_highlight](https://pub.dev/packages/re_highlight) - The syntax highlighting engine used by CodeForge (Rust-based regex port).
*   **markdown_widget**: [pub.dev/packages/markdown_widget](https://pub.dev/packages/markdown_widget) - Used for rendering hover documentation.

## Alternatives
*   **code_text_field**: [pub.dev/packages/code_text_field](https://pub.dev/packages/code_text_field) - Popular but less performant on large files.
*   **flutter_code_editor**: [pub.dev/packages/flutter_code_editor](https://pub.dev/packages/flutter_code_editor) - Another strong contender, often used in web-based Flutter IDEs.

## Related Concepts
*   **Language Server Protocol (LSP)**: [microsoft.github.io/language-server-protocol](https://microsoft.github.io/language-server-protocol/) - The standard used for intelligence.
*   **Rope Data Structure**: [en.wikipedia.org/wiki/Rope_(data_structure)](https://en.wikipedia.org/wiki/Rope_(data_structure)) - The algorithm behind the performance.
