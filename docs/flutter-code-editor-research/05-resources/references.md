# Resources

## Official
*   **Pub.dev**: [flutter_code_editor](https://pub.dev/packages/flutter_code_editor)

## Dependencies
*   **highlight**: The Dart port of highlight.js. Understanding this package helps with language definitions.
*   **flutter_highlight**: Provides the theme maps.

## Alternatives
*   **code_forge**: Better for performance/LSP, but no Web support.
*   **code_text_field**: The predecessor to this package (forked from it).
